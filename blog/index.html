hexo常用指令

hexo new '文章名稱' 在Hexo部落格中新增貼文

hexo new page '分類名稱' 在hexo部落格中新增一個文章的分類項目 (關於我們/聯絡方式/最新消息等等)
若要在主畫面中顯示新增的分類，需要從themes中設定模板的樣式，在模板中新增連結(nav或sidebar中)

hexo generate 在專案文件中生成Public資料夾，會將hexo部落格中內容生成為一般的html格式，用以在其他空間部屬網站時使用

hexo clean 將目前啟動中的server清理暫存重啟，當網站遇到未知的bug時(調整config後網站並未應用更改後的設定等等)，會使用此指令重整部落格網頁

hexo deploy 部屬github page時使用

hexo安裝手冊

1. 安裝node.js / git

cli輸入node -v / npm -v可以確認是否有安裝成功，若有安裝成功會顯示目前使用的版本

2. npm install -g hexo-cli

hexo -v查看hexo跟其使用套件的版本

3. 在要建立專案的位置 hexo init '英文網站名稱'

4. hexo server 將server啟動

5. hexo new '文章標題' 在部落格新增文章

6. 新增的文章會在source/posts中以md文件的形式存在，編輯內容也是使用md格式
