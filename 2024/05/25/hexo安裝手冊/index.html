hexo安裝手冊

1. 安裝node.js / git

cli輸入node -v / npm -v可以確認是否有安裝成功，若有安裝成功會顯示目前使用的版本

2. npm install -g hexo-cli

hexo -v查看hexo跟其使用套件的版本

3. 在要建立專案的位置 hexo init '英文網站名稱'

4. hexo server 將server啟動

5. hexo new '文章標題' 在部落格新增文章

6. 新增的文章會在source/posts中以md文件的形式存在，編輯內容也是使用md格式
